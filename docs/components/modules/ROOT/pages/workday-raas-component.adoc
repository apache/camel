[[workday-component]]
= Workday Component
:page-source: components/camel-workday/src/main/docs/workday-raas-component.adoc

*Since Camel 2.19*

// HEADER START
*Only producer is supported*
// HEADER END

The *Workday*: components provides the ability to detect and parse documents with 
Workday. 

In order to use the Workday component, Maven users will need to add the
following dependency to their `pom.xml`:

*pom.xml*

[source,xml]
----
<dependency>
    <groupId>org.apache.camel</groupId>
    <artifactId>camel-workday</artifactId>
    <version>x.x.x</version>
    <!-- use the same version as your Camel core version -->
</dependency>
----
