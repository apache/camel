== Example usage

The `recipientList` EIP can utilize a variable:

[tabs]
====

Java::
+
[source,java]
----
from("direct:a").recipientList(variable("myVar"));
----

XML::
+
[source,xml]
----
<route>
  <from uri="direct:a" />
  <recipientList>
    <variable>myVar</variable>
  </recipientList>
</route>
----

YAML::
+
[source,yaml]
----
- route:
    from:
      uri: direct:a
      steps:
        - recipientList:
            expression:
              variable:
                expression: myVar
----
====

In this case, the list of recipients are contained in the variable
'myVar'.
