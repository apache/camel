https://github.com/jamesagnew/hapi-fhir/blob/master/hapi-fhir-base/src/main/java/ca/uhn/fhir/parser/XmlParser.java[HAPI-FHIR's]
XML parser to parse to/from XML format to/from a HAPI-FHIR's `IBaseResource`.
