*Since Camel {since}*

*{component-header}*

The Azure Event Grid component provides the capability to publish events to
https://learn.microsoft.com/en-us/azure/event-grid/overview[Azure Event Grid] topics
using https://learn.microsoft.com/en-us/azure/event-grid/cloud-event-schema[CloudEvents].

Azure Event Grid is an eventing service for the cloud that enables event-driven architectures.
It provides event routing from any source to any destination with rich filtering capabilities.

*Prerequisites*

You must have a valid Microsoft Azure account and an Event Grid topic.
More information is available at the https://learn.microsoft.com/en-us/azure/event-grid/[Azure Event Grid documentation].

Maven users will need to add the following dependency to their `pom.xml`
for this component:

[source,xml]
----
<dependency>
    <groupId>org.apache.camel</groupId>
    <artifactId>camel-azure-eventgrid</artifactId>
    <version>x.x.x</version>
    <!-- use the same version as your Camel core version -->
</dependency>
----

== Authentication

The component supports the following credential types via the `credentialType` option:

ACCESS_KEY:: Uses the Event Grid topic access key for authentication (default).
Set the `accessKey` option with your topic key.
TOKEN_CREDENTIAL:: Uses a `TokenCredential` implementation (e.g., Azure Identity `DefaultAzureCredential`).
Set the `tokenCredential` option.
AZURE_IDENTITY:: Uses Azure Identity default credential chain automatically.

== Usage

=== Sending events

The producer publishes events as CloudEvents to the configured Event Grid topic endpoint:

[tabs]
====
Java::
+
[source,java]
----
from("direct:start")
    .to("azure-eventgrid:https://my-topic.eastus-1.eventgrid.azure.net/api/events?accessKey=RAW(myKey)");
----

XML::
+
[source,xml]
----
<route>
    <from uri="direct:start"/>
    <to uri="azure-eventgrid:https://my-topic.eastus-1.eventgrid.azure.net/api/events?accessKey=RAW(myKey)"/>
</route>
----

YAML::
+
[source,yaml]
----
- route:
    from:
      uri: direct:start
    steps:
      - to:
          uri: azure-eventgrid:https://my-topic.eastus-1.eventgrid.azure.net/api/events
          parameters:
            accessKey: "RAW(myKey)"
----
====
