*Since Camel {since}*

*{component-header}*

The camel Atomix-Set component allows you to work with http://atomix.io[Atomix's] https://atomix.io/docs/latest/user-manual/primitives/DistributedSet/[Distributed Set] collection.

Maven users will need to add the following dependency to their pom.xml
for this component:

[source,xml]
----
    <dependency>
        <groupId>org.apache.camel</groupId>
        <artifactId>camel-atomix</artifactId>
        <version>${camel-version}</version>
    </dependency>
----

== Examples

=== Consumer Example

Consuming from an Atomix Set

[source,java]
----
from("atomix-set:set")
  .to("mock:result");
----

=== Producer Example

Send a message to Atomix Set

[source,java]
----
from("direct:start")
  .to("atomix-set:set");
----
