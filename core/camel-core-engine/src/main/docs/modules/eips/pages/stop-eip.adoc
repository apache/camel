== Using Stop

We want to stop routing a message if the message body contains the word Bye.
In the xref:choice-eip.adoc[Content Based Router] below we use `stop` in such case.

[source,java]
----
from("direct:start")
    .choice()
        .when(body().contains("Hello")).to("mock:hello")
        .when(body().contains("Bye")).to("mock:bye").stop()
        .otherwise().to("mock:other")
    .end()
.to("mock:result");
----

And in XML:

[source,xml]
----
<route>
  <from uri="direct:start"/>
  <choice>
    <when>
      <simple>${body} contains 'Hello'</simple>
      <to uri="mock:hello"/>
    </when>
    <when>
      <simple>${body} contains 'Bye'</simple>
      <stop/>
    </when>
    <otherwise>
      <to uri="mock:other"/>
    </otherwise>
  </choice>
</route>
----

=== Calling stop from Java

You can also mark an `Exchange` to stop being routed from Java with the following code:

[source,java]
----
Exchange exchange = ...
exchange.setRouteStop(true);
----
