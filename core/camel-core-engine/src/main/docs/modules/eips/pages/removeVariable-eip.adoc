== Example

We want to remove a variable with key "myVar" from the exchange:

[tabs]
====
Java::
+
[source,java]
----
from("seda:b")
  .removeVariable("myVar")
  .to("mock:result");
----

XML::
+
[source,xml]
----
<route>
  <from uri="seda:b"/>
  <removeVariable name="myVar"/>
  <to uri="mock:result"/>
</route>
----
====

TIP: If you want to remove all variables from the `Exchange` then use `*` as the name.
