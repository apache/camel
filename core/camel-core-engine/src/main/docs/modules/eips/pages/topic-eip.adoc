= Topic EIP
:doctitle: Topic
:shortname: topic
:description: Topic which sends to all destinations.
:since: 
:supportlevel: Stable
:tabs-sync-option:

Topic mode for the xref:loadBalance-eip.adoc[Load Balancer] EIP.
With this policy, then all destinations are selected.

== Examples

In this example, we send the message to all three endpoints:

[tabs]
====

Java::
+
[source,java]
----
from("direct:start")
    .loadBalance().topic()
        .to("seda:x")
        .to("seda:y")
        .to("seda:z")
    .end();
----

XML::
+
[source,xml]
----
<route>
<from uri="direct:start"/>
    <loadBalance>
       <topic/>
       <to uri="seda:x"/>      
       <to uri="seda:y"/>      
       <to uri="seda:z"/>       
    </loadBalance> 
</route>
----
